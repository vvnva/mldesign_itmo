# Классификация клиентских обращений OTRS

Репозиторий проекта по автоматической обработке письменных обращений клиентов сети АЗС.

## Цель

Автоматизировать первичную маршрутизацию и типизацию обращений по тексту сообщения для сокращения сроков обработки и снижения нагрузки на операторов.

## Основные задачи

1. Классификация обращения по:
   - **Типу** (`Консультация`, `Заявка`, `Жалоба`, `Пожелание`, `Благодарность`, `Дополнение к тикету`)
   - **Теме** (`Общая тема`)
   - **Тематике** (детализированная категория внутри темы)

2. Побочная задача:
   - **Выделение сущностей (NER)**: номера АЗС, номера карт лояльности для ускорения маршрутизации и дополнительной валидации классификации
